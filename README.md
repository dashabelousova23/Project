# Project.
## Игра "Химические элементы"
## Платформа: Unity (C#).
## Проблема: Низкая вовлеченность детей при изучении химии через классические учебники.
## Решение: Обучающая 3D-игра в сеттинге Средневековья, где запоминание элементов происходит через игровой процесс.
## Игровая механика: Игрок управляет лучником, исследует замок и проходит уровни-комнаты, решая химические задачи для продвижения дальше и получения награды.
## Уникальность: Сочетание исторической атмосферы и проверки знаний через активные действия (стрельбу по мишеням-элементам (щитам)), а не сухие тесты.
## Результат: Рабочий прототип игры, включающий систему передвижения, стрельбы и несколько тематических уровней.
